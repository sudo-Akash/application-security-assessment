# PROJECT OVERVIEW

## Project name
OWASP Juice Shop

## Project description
OWASP Juice Shop is an intentionally vulnerable web application developed by OWASP for security awareness, training, and application security testing. The project simulates real-world security vulnerabilities commonly found in modern web applications.

## Repository information
Repository URL: https://github.com/juice-shop/juice-shop

## Assessment objective
The objective of this assessment is to perform a comprehensive application security review through:

- Manual secure code review
- Static Application Security Testing (SAST)
- Software Supply Chain Analysis
- Vulnerability Identification and Risk Assessment
- Remediation Recommendation Development

## Project structure

### Key directories

- frontend/ – User interface components
- routes/ – Application routes and API endpoints
- models/ – Database models and business logic
- lib/ – Utility functions and shared libraries
- data/ – Application data and configuration
- config/ – Application configuration files

## Scope of assessment

The security assessment focuses on:

- Authentication and Authorization
- Input Validation
- Injection Vulnerabilities
- Business Logic Security
- Error Handling
- Cryptographic Implementation
- Dependency Security
- Software Supply Chain Risks

## Expected deliverables

1. Project Overview Report
2. Manual Code Review Report
3. SAST Scan Report
4. Supply Chain Security Report
5. Final Application Security Assessment Report

## Assessment environment

All analysis, documentation, and reporting activities are performed within the ArcForge development environment as required by the project guidelines.
